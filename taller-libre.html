Constructor de obra

Pulsa Agregar para crear tu primer elemento: espacio, edificio, malacate, torre grúa o grúa pluma. Luego haz clic sobre él para ver su ficha, arrástralo para ubicarlo, gíralo o elimínalo.
Controles: Arrastrar (clic izq. en vacío) = rotar · Clic derecho = mover cámara · Rueda = zoom
Elementos: clic y arrastrar para mover · clic corto = ficha
Torre grúa / pluma: define el largo del brazo y el radio de giro; el brazo gira solo
